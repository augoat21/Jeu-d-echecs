# Chess Piece Assets

Place your chess piece images here.

## Expected file structure:
```
pieces/
├── wK.png  (White King)
├── wQ.png  (White Queen)
├── wR.png  (White Rook)
├── wB.png  (White Bishop)
├── wN.png  (White Knight)
├── wP.png  (White Pawn)
├── bK.png  (Black King)
├── bQ.png  (Black Queen)
├── bR.png  (Black Rook)
├── bB.png  (Black Bishop)
├── bN.png  (Black Knight)
└── bP.png  (Black Pawn)
```

## Free piece sets:
- https://commons.wikimedia.org/wiki/Category:SVG_chess_pieces
- https://github.com/lichess-org/lila/tree/master/public/piece

If no images are provided, the game will use Unicode chess symbols.
